# password-generator

Создать генератор паролей с графическим интерфейсом и компилировать его в .exe для удобства использования в работе

Генератора пароле удобный инструмент для работы с информационными системами и пользователями


технологии импользованные в программе:
редактор исходного кода - Visual Studio Code (VS Code)
библиотеки customtkinter, tkinter, random.
инструмент для упаковки Python-приложений - PyInstaller
